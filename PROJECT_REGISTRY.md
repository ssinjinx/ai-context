## Status Key

- `active` — Currently being worked on
- `wip` — Work in progress, paused
- `maintenance` — Bug fixes only
- `archived` — Done, not touching
- `idea` — Not started yet

## Adding New Projects

When creating a new project:

1. Check this file first — does it already exist?
2. If yes → use existing path/URL
3. If no → create project, then add entry here
4. Update the `Last Updated` timestamp

## Template for New Entry

```markdown
| project-name | owner/repo | ~/projects/project-name | status | Brief description | YYYY-MM-DD |
```
